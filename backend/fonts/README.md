# Fonts Directory

This directory contains fonts used for PDF generation with Vietnamese support.

## Roboto-Regular.ttf

Roboto is an open-source font from Google that supports Vietnamese characters.

### Download Font

If the font is missing, run:

```bash
node scripts/downloadFont.js
```

Or download manually from [Google Fonts - Roboto](https://fonts.google.com/specimen/Roboto) and place the `Roboto-Regular.ttf` file in this directory.
